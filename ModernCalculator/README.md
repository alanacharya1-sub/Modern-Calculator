# Modern Calculator

A fully functional scientific calculator with graphing capabilities, built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations (+, -, ×, ÷, %)
- Scientific functions (sin, cos, tan, log, etc.)
- Graphing capabilities
- Multiple calculator modes (Scientific, Programmer, Matrix, Statistics, Graph)
- Calculation history
- Light/Dark themes
- Keyboard shortcuts

## Getting Started

1. Open `index.html` in your web browser
2. Start calculating!

## Keyboard Shortcuts

- `0-9`: Numbers
- `+`, `-`, `*`, `/`: Basic operations
- `Enter` or `=`: Calculate
- `Backspace`: Delete last character
- `Escape`: Clear display
- `(`, `)`: Parentheses
- `^`: Exponentiation
- `Ctrl+P`: Insert π
